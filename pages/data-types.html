PostgreSQL Data Types

Numeric Data Types

Data Type Description Size
SMALLINT Small-range integer 2 bytes
INTEGER Typical choice for integer 4 bytes
BIGINT Large-range integer 8 bytes
DECIMAL User-specified precision variable
NUMERIC User-specified precision variable
REAL Variable-precision 4 bytes
DOUBLE PRECISION Variable-precision 8 bytes

Character Data Types

Data Type Description
CHAR(n) Fixed-length character string
VARCHAR(n) Variable-length character string
TEXT Variable unlimited length

Date/Time Data Types

Data Type Description
DATE Date (no time of day)
TIME Time of day (no date)
TIMESTAMP Date and time
INTERVAL Time interval

Boolean Data Type

-- Boolean can be: true, false, or null
CREATE TABLE example (
id SERIAL PRIMARY KEY,
is_active BOOLEAN DEFAULT true
);

Special Data Types

  • UUID: Universally unique identifiers
  • JSON/JSONB: JSON data with/without indexing
  • ARRAY: Array of any data type
  • BYTEA: Binary data
  • MONEY: Currency amount

Choosing the Right Data Type:

  • Consider the range of values needed
  • Think about storage requirements
  • Consider performance implications
  • Think about data validation needs
