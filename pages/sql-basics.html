SQL Basics

What is SQL?

SQL (Structured Query Language) is the standard language for managing and manipulating relational databases. It's used for storing, manipulating, and retrieving data in databases.

Basic SQL Commands

DDL (Data Definition Language)

-- Create a table
CREATE TABLE users (
id SERIAL PRIMARY KEY,
name VARCHAR(100),
email VARCHAR(255)
);

-- Modify table structure
ALTER TABLE users ADD COLUMN age INTEGER;

-- Delete table
DROP TABLE users;

DML (Data Manipulation Language)

-- Insert data
INSERT INTO users (name, email) VALUES ('John', 'john@example.com');

-- Update data
UPDATE users SET age = 25 WHERE name = 'John';

-- Delete data
DELETE FROM users WHERE name = 'John';

DQL (Data Query Language)

-- Select all columns
SELECT * FROM users;

-- Select specific columns
SELECT name, email FROM users;

-- Filter results
SELECT * FROM users WHERE age > 21;

-- Sort results
SELECT * FROM users ORDER BY name ASC;

SQL Operators

  • Comparison: =, >, <, >=, <=, <>
  • Logical: AND, OR, NOT
  • Special: BETWEEN, IN, LIKE, IS NULL

SQL Functions

  • Aggregate: COUNT(), SUM(), AVG(), MAX(), MIN()
  • String: CONCAT(), SUBSTRING(), UPPER(), LOWER()
  • Date: NOW(), DATE(), YEAR(), MONTH(), DAY()

Best Practices:

  • Use meaningful table and column names
  • Always use proper data types
  • Include appropriate constraints
  • Write clear and maintainable queries
  • Use indexes for better performance
