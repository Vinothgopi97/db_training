Basic Database Concepts

What is a Database?

A database is an organized collection of structured information or data, typically stored electronically in a computer system. Databases are designed to manage large collections of information and provide fast, accurate, and efficient data retrieval.

Key Terms

  • Data: Raw facts and figures that can be processed to produce information
  • Database Management System (DBMS): Software that manages databases
  • Table: A collection of related data organized in rows and columns
  • Record/Row: A single entry in a table
  • Field/Column: A single piece of information in a record
  • Primary Key: A unique identifier for each record

Example of a Simple Table

ID (Primary Key) Name Age City
1 John Doe 25 New York
2 Jane Smith 30 London

Types of Databases

  • Relational Databases: Data organized in tables with relationships between them (e.g., PostgreSQL, MySQL)
  • NoSQL Databases: Non-relational databases for unstructured data (e.g., MongoDB)
  • Object-Oriented Databases: Data stored as objects
  • Graph Databases: Data stored in nodes and edges

Why Relational Databases?
Relational databases are the most widely used type of database because they:

  • Ensure data integrity
  • Support complex queries
  • Provide transaction management
  • Maintain data consistency
